# EyeKotlin
> Kotlin作为安卓的官方开发语言, 还是要学一下的 在过年时花了一个星期看了下文档, 也写了一些小DEMO, 感觉是时候开始写一些比较完整的项目了

## 架构
1. 采用主流MVP + Retrofit + RxJava进行开发
2. 单Activity + 多Fragment
